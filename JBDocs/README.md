# JB Docs

This folder contains working notes for understanding `MH2p_SD_ModKit`, especially the parts that are opaque in the main repo such as bundled binaries, generated package manifests, and external dependencies.

## Docs

- [repo-orientation.md](repo-orientation.md) - high-level map of what is in this repo and how the pieces fit together
- [binary-sources-and-build-notes.md](binary-sources-and-build-notes.md) - what is known about bundled binaries and generated update metadata, what appears to come from elsewhere, and what still needs to be verified

## Current status

- The repo-authored logic is mostly shell-script based.
- The packaged update metadata under `Meta/` and `Data/*/installer.txt` appears to be generated by an external tool named `mnfc`.
- The bundled `fecswap` file is an ARM ELF binary tracked directly in git, but its source code is not present in this repository.
- The runtime also depends on device-native binaries that are not included here, such as `showimage`, `pc`, and the original `servicemgrmibhigh`.
